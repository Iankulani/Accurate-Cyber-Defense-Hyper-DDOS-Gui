# Accurate-Cyber-Defense-Hyper-DDOS-Gui
Accurate-Cyber-Defense-Hyper-DDOS-GUI is a lightweight defender-focused application designed to help security teams visualize, simulate, and mitigate distributed denial-of-service threats within controlled, lawful environments.


## How to clonw the repo
```bash
git clone https://github.com/Iankulani/Accurate-Cyber-Defense-Hyper-DDOS-Gui.git

cd Accurate-Cyber-Defense-Hyper-DDOS-Gui
```

python Accurate-Cyber-Defense-Hyper-DDOS-Gui.py
